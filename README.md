# servlet-jdbc-demo

## Iteration 1: Simple Servlets w/o database

```
mvn clean compile
java -jar servlet-jdbc-demo-1.0-SNAPSHOT-shaded.jar
```


## Iteration 2: Add postgres

```
./run-prostgres.sh
```
